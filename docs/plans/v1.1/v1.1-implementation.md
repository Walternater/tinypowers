### 实现细节

**check-compile.sh**：
```bash
#!/bin/bash
set -e

if [ -f "pom.xml" ]; then
  mvn compile -q -DskipTests
elif [ -f "build.gradle" ]; then
  ./gradlew compileJava --quiet
else
  echo "ERROR: 未知构建工具"
  exit 1
fi

echo "✓ 编译通过"
```

**check-style.sh**：
```bash
#!/bin/bash

if [ -f "pom.xml" ]; then
  mvn spotless:check -q || {
    echo "✗ 格式化检查失败"
    echo "修复: mvn spotless:apply"
    exit 1
  }
fi

echo "✓ 格式化检查通过"
```
