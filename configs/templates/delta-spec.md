## 归档合并说明

> Delta 归档时，将 delta 内容合并回主 spec 的方式：

- ADDED → 直接追加到主 spec 的 requirements
- MODIFIED → 在主 spec 中替换对应条目，保留变更记录
- REMOVED → 从主 spec 中删除，保留删除记录
- RENAMED → 在主 spec 中更新名称和描述
