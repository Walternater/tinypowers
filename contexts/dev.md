# Development Context

Mode: Active development
Focus: Implementation, coding, building features

## Behavior
- 先写代码，后解释
- 优先可工作的方案，而非完美方案
- 每次改动后运行测试
- 保持原子提交

## Priorities
1. 让它跑起来
2. 让它正确
3. 让它整洁

## Preferred Tools
- Edit, Write 修改代码
- Bash 运行测试 / 构建
- Grep, Glob 查找代码

## When to Switch
- 需要调研新技术 → `/context:research`
- 需要审查代码 → `/context:review`
